# Canvas Extension App

Canvas Extension App is a Streamlit planning tool that connects to Canvas, pulls upcoming assignments and calendar events, calculates priority-based study blocks, and can add or remove generated study blocks on the user's personal Canvas calendar.

## Features

- Connect to Canvas with a manual access token.
- Select which Canvas calendars to sync.
- Fetch assignments and existing calendar events for the next 14 days.
- Retrieve assignment points and gradebook category weights from Canvas assignment groups.
- Edit fetched assignment and event data before scheduling.
- Include, exclude, or remove specific assignments and events from scheduler calculations.
- Generate study blocks around existing busy calendar events.
- Add generated study blocks to the user's Canvas personal calendar.
- Delete study blocks previously generated by this app.

## How The App Works

1. The user enters a Canvas manual access token in the Streamlit sidebar.
2. `app.py` creates a `CanvasInterface` and loads available Canvas calendars.
3. The user selects calendars and clicks `Fetch Canvas Data`.
4. `CanvasInterface` retrieves assignments, events, points, courses, due dates, assignment groups, and gradebook category weights.
5. `Dashboard.py` shows editable assignment and event tables.
6. The user edits or excludes rows as needed.
7. `app.py` converts the edited session data into scheduler input.
8. `SmartScheduler` generates study blocks based on due date urgency, assignment points, gradebook category weight, and existing calendar events.
9. The user can add the generated study blocks to Canvas or delete generated study blocks from Canvas.

## Shared Data Shape

Fetched assignment dictionaries are normalized before they reach the dashboard and scheduler. Important fields include:

```python
{
    "id": "assignment_<course>_<assignment>",
    "include": True,
    "canvas_assignment_id": 123,
    "context_code": "course_456",
    "course_id": 456,
    "course_name": "Course Name",
    "title": "Assignment Name",
    "due_at": "2026-05-10T23:59:00Z",
    "points": 100,
    "assignment_group_name": "Projects",
    "group_weight": 0.2,
    "group_weight_percent": 20.0,
}
```

Fetched event dictionaries use this shape:

```python
{
    "id": "event_<canvas_event_id>_<start>",
    "include": True,
    "canvas_event_id": 789,
    "context_code": "user_123",
    "title": "Class Meeting",
    "start": "2026-05-10T10:00:00Z",
    "end": "2026-05-10T11:00:00Z",
    "context": "Personal",
}
```

Generated study blocks are shaped for Canvas calendar creation:

```python
{
    "title": "study block for Essay CPSC 362",
    "start_at": "2026-05-06T08:00:00",
    "end_at": "2026-05-06T09:00:00",
    "calendar_id": "user_123",
    "assignment_name": "Essay",
    "course_name": "CPSC 362",
    "canvas_assignment_id": 456,
    "description": "Priority score: ...",
}
```

Start the Streamlit app:

```bash
.venv/bin/streamlit run app.py
```

Then open the local URL shown by Streamlit, usually:

```text
http://localhost:8501
```

## Canvas Token Setup

To create a Canvas access token:

1. Open Canvas.
2. Go to `Account`.
3. Open `Settings`.
4. Find `Approved Integrations`.
5. Create a new access token.
6. Copy the token and paste it into the app sidebar.

The current app is configured for:

```python
https://csufullerton.instructure.com
```

This value is set in `CANVAS_BASE_URL` inside `app.py`.

## Scheduler Logic

The scheduler prioritizes assignments using:

- Time until due date
- Assignment points
- Assignment group/category weight from Canvas
- Existing busy calendar events

The scheduler avoids existing events and adds a buffer between study blocks. Study blocks are currently one hour long and are generated within normal study hours.

## Canvas Calendar Sync

After running the optimizer, the Smart Schedule tab provides two Canvas actions:

- `Add study blocks to Canvas`: creates the generated study blocks on the user's personal Canvas calendar.
- `Delete generated study blocks from Canvas`: deletes study blocks generated by this app in the current schedule window.

Generated study blocks are marked with this description marker:

```text
Generated by Smart Canvas Planner
```

Deletion uses this marker and the title prefix `study block for` so the app does not intentionally delete unrelated Canvas calendar events.

## Known Limitations

- Real Canvas add/delete behavior requires a valid token and live Canvas account.
- The Canvas base URL is currently hardcoded for CSUF.
- The app stores fetched data in Streamlit session state; refreshing or clearing the session removes local edits.
